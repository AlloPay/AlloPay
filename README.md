# AlloPay

## Configuration

- See .env.template
- app/play-service-account.secret.json

## Setup

```bash
yarn
```

## API

### Local

```bash
# Build
yarn api build

# Test
yarn api test

# Run
yarn api start
```


## App

```bash
# Build
yarn app build

# Test
yarn app test

# Run
yarn app start
```

## Contracts

```bash
# Build
yarn contracts build

# Test
yarn contracts test
```
