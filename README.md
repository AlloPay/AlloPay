# Zallo

## Setup

- See `.env.template` for required configuration

```bash
yarn
```

## API

### Local

```bash
# Build
yarn api build

# Test
yarn api test

# Run
yarn api start
```


## App

```bash
# Build
yarn app build

# Test
yarn app test

# Run
yarn app start
```

## Contracts

```bash
# Build
yarn contracts build

# Test
yarn contracts test
```
